# 基于统一几何构型与非线性耦合特征的参数体系

# 1. 几何构型的统一化描述与等效参数推导

实现多截面承载力统一预测的首要前提是建立一个能够涵盖所有目标形状（圆形、方形、矩形、圆端形）的通用几何描述体系。传统的分类方法将不同形状视为离散对象，而统一理论则将其视为连续几何变换的特例。

# 1.1 广义“典型截面”的参数定义

根据Lin等（2021）的研究，所有常见CFST截面均可视为一个带有圆角的矩形截面（Typical Form）的特例 1。相关参数解释如下：

截面宽度 $( b )$ 与深度 $( h )$ ：定义为截面的外部轮廓尺寸。设定 $b \geq h _ { * }$ 。对于圆形和方形，$b { = } h _ { \circ }$ 。

钢管外圆角半径 $( r _ { 0 } )$ ：这是控制截面形状演变的关键参数。

当 $r _ { 0 } { = } 0$ 时，截面为矩形或方形（存在应力集中，角部约束强，边部约束弱）。

当 $r _ { 0 } { = } h / 2$ 时，截面为圆形或圆端形（约束分布更均匀）。

钢管壁厚(t)：直接决定含钢率和局部屈曲性能。

这种参数化方法将“形状”这一离散概念转化为了连续变量 $r _ { 0 } / h$ （角径比）和 $b / h$ （高宽比）。

# 1.2 等效圆柱的物理换算

采用面积守恒原则进行物理换算1。该原则要求等效圆柱必须同时满足两个条件：

1. 等效核心混凝土面积 $( A _ { c , e q } )$ 等于原截面混凝土面积 $( A _ { c } ) _ { \circ }$

2. 等效钢管面积 $( A _ { s , e q } )$ 等于原截面钢管面积 $( A _ { s } ) _ { \circ }$ 。

参数推导来源与公式：

基于 Lin et al.，原截面的混凝土面积 $A _ { c }$ 和钢管面积 $A _ { s }$ 分别为：

$$
A _ {c} = (b - 2 t) (h - 2 t) - (4 - \pi) r _ {1} ^ {2}
$$

$$
A _ {s} = 2 t (b + h) - 4 t ^ {2} - (4 - \pi) \left(r _ {0} ^ {2} - r _ {1} ^ {2}\right)
$$

由此推导出的等效物理参数为：

等效混凝土半径 $( R _ { e } )$ ：

$$
R _ {e} = \sqrt {\frac {A _ {c}}{\pi}} = \sqrt {\frac {(b - 2 t) (h - 2 t) - (4 - \pi) r _ {1} ^ {2}}{\pi}}
$$

等效钢管厚度 $( t _ { e } )$ ：

$$
t _ {e} = \sqrt {\frac {A _ {c} + A _ {s}}{\pi}} - R _ {e}
$$

# 1.3 核心约束机理参数：约束有效性系数 $( k _ { e } )$

虽然等效圆柱在材料量上等效，但矩形或圆端形钢管对混凝土的侧向约束能力远低于圆形钢管。圆形钢管产生连续的环向拉力，提供均匀的径向压力；而矩形钢管的平直边在侧压力下发生弯曲变形，导致中部混凝土约束极弱，仅角部存在有效约束。为了在模型中物理量化这一差异，必须引入约束有效性系数 $( k _ { e } )$ ）。

# 1.3.1 拱作用模型与有效约束面积

非圆形截面中的约束机制遵循“拱作用”（Arching Action）理论。在钢管壁的平直段，钢管刚度较小，无法有效约束混凝土的横向膨胀；而在角部，钢管刚度大，约束强。因此，有效约束区呈拱形分布在截面角部，而平直段与拱形之间的区域被称为“非有效约束区”。

参数推导：

Lin 等（2021）基于有限元分析结果，提出了基于截面几何特征的抛物线拱模型。

有效约束面积 $A _ { c c }$ 定义为总混凝土面积减去非有效区域面积。

非有效区域的分布取决于抛物线的切线斜率，该斜率计算如下：

$$
\tan \alpha = \frac {h - 2 t}{b - 2 r _ {1} - 2 t}
$$

$$
\tan \beta = \frac {b - 2 t}{h - 2 r _ {1} - 2 t}
$$

这些正切值物理上代表了截面长宽比对拱形“平缓度”的影响。截面越扁平（b / h 越大），tan $\alpha$ 越小，非有效区域面积越大。

# $1 . 3 . 2 ~ k _ { e }$ $k _ { e }$ 的统一计算公式

基于上述几何推导，统一的约束有效性系数 $k _ { e }$ 被定义为有效约束面积与总混凝土面积之比：

$$
k _ {e} = \frac {A _ {c c}}{A _ {c}} = 1 - \frac {\text {非 有 效 区 域 总 面 积}}{A _ {c}}
$$

$$
k _ {e} = 1 - \frac {\tan \alpha (b - 2 r _ {1} - 2 t) ^ {2} + \tan \beta (h - 2 r _ {1} - 2 t) ^ {2}}{3 (b - 2 t) (h - 2 t) - 3 (4 - \pi) r _ {1} ^ {2}}
$$

参数影响分析：

- 对于圆形柱： $r _ { 0 } { = } h / 2 , b { = } h$ ，公式中的分子项（平直段长度）归零，计算得 $k _ { e } = 1 . 0$ 。这在物理上意味着全截面有效约束1。

- 对于方形柱： $r _ { 0 } { = } 0 , b { = } h$ ，该系数显著小于1（通常在0.3-0.5之间），反映了方形截面约束效率的低下。

- 对于圆端形柱：该公式能够自动捕捉圆弧段（高效约束）与平直段（低效约束）的混合效应。随着平直段长度增加， $k _ { e }$ 值降低。

# 2. 材料参数 $( \xi , \sigma _ { \mathfrak { R } } )$

在确定了等效几何和约束效率后，需要引入描述材料非线性行为及钢-混相互作用强度的物理参数。

# 2.1 套箍系数 (ξ)

由韩林海（Han）等提出，广泛用于中国规范（GB 50936）及相关理论研究。

定义公式：

$$
\xi = \frac {A _ {s} f _ {y}}{A _ {c} f _ {c}}
$$

引入原因： $\xi$ 反映了钢管对混凝土的“名义约束潜力”。它是一个综合指标，结合了含钢率（ $( A _ { s } / A _ { c } )$ ）和强度比 $( f _ { y } / f _ { c } )$ ）。 $\xi$ 值越大，意味着钢管相对越强，能提供的侧向约束力越大，混凝土强度的提高幅度也越大。

# 2.2 有效侧向应力 $( \sigma _ { \mathfrak { R } } )$

为了更精细地预测承载力，单纯使用 $\xi$ 可能不够，因为它未包含形状效应。Lin 等（2021）提出的有效侧向应力参数结合了形状因子与材料强度 1。

公式：

$$
\sigma_ {\Re} = k _ {e} \cdot \sigma_ {r} = k _ {e} \cdot \left(\frac {2 t _ {e}}{D _ {e q} - 2 t _ {e}} \sigma_ {s h}\right)
$$

$$
\sigma_ {r} = \frac {2 t _ {e}}{2 R _ {e}} f _ {y} = \frac {t _ {e}}{R _ {e}} f _ {y}
$$

引入原因：这是核心混凝土实际“感受”到的平均侧向约束力。它将几何形状（通过 $k _ { e . }$ ）、钢管厚度 $( t _ { e } )$ ）和钢管环向应力（ $( \sigma _ { s h } )$ ）统一为一个应力指标。

# 3. 截面形状强度修正系数 $\left( \varphi _ { r } , \varphi _ { a } \right)$

虽然等效圆柱 $( R _ { e } , t _ { e } )$ 和约束系数 $( k _ { e } )$ 解决了本构层面的统一，但在宏观承载力预测（设计公式层面）上，直接引入形状修正系数可以大幅简化模型学习难度。Lin 等（2021）文中提炼出了两个关键的修正参数。

# 3.1 半径比影响系数 $\left( \varphi _ { r } \right)$

来源：Lin等（2021）基于参数化分析拟合1。

定义：

$$
\varphi_ {r} = 1 + 0. 0 0 3 f _ {c} \left[ 1 - \exp \left(r _ {0} / h - 0. 5\right) \right]
$$

# 引入原因：

该参数量化了角部圆角化对承载力的影响。

对于普通强度混凝土， $\varphi _ { r } \approx 1 . 0$ 。这揭示了：增大圆角半径虽然减小了截面面积（负面影响），但增强了约束效应（正面影响），两者相互抵消。

对于高强混凝土，由于其对约束不敏感，面积减小的负面影响占主导，导致 $\varphi _ { r }$ 随圆角增大而减小。

# 3.2 高宽比影响系数 $( \varphi _ { a } )$

来源：Lin 等（2021）1。

定义：

$$
\varphi_ {a} = 1 + \left(0. 0 0 2 f _ {c} + 0. 6 2\right) (b / h - 1)
$$

# 引入原因：

该参数量化了截面扁平化带来的承载力变化。随着 b / h 增加（矩形越扁），尽管面积可能增加，但约束效率急剧下降。该系数将这种几何变化转化为承载力的线性修正。

# 4. 全局稳定性参数：长细比（λ)与相对长细比（λ）

CFST 柱的破坏模式主要分为截面强度破坏（压溃）和整体失稳破坏（弯曲）。对于短柱，承载力由材料强度控制；而对于长柱，承载力则受限于刚度。基础参数中的构件长度 L 和截面尺寸 b, h 虽然隐含了这一信息，但机器学习模型很难直接从这些线性输入中“学习”到非线性的屈曲规律（即承载力与长度的平方成反比）。因此，必须引入表征整体稳定性的无量纲参数。

# 4.1 长细比（λ）

长细比是结构力学中最基础的稳定性参数，广泛出现于AISC 360-16、Eurocode 4及GB 50936等规范中。其定义公式为：

$$
\lambda = \frac {K L}{r}
$$

其中：

K ——有效长度系数，取决于柱端的约束条件（如两端铰接 $\mathrm { K } { = } 1 . 0$ ，两端固接$\mathrm { K } { = } 0 . 5$ ）。

L——为构件的几何长度。

r——为组合截面的回转半径， $r { = } \sqrt { I / A }$ 。

引入原因：

一、直接使用 L 作为输入特征存在尺度敏感性问题。一个长度为 3 米的柱子，对于大截面可能是短柱，对于小截面则是长柱。 $\lambda$ 通过将长度与截面几何刚度进行归一化，消除了尺寸效应，直接反映构件的柔度。在 ML 模型中，引入 $\lambda$ 可以帮助算法快速区分构件属于“短柱区间”（强度控制）还是“长柱区间”（稳定控制），从而在决策树或神经网络中激活不同的权重路径。

二、长细比直接决定了欧拉临界力（ ${ ' } P _ { \cal E } { = } { \pi } ^ { 2 } E I / { \left( K L \right) } { \rangle }$ ）的大小。

当 λ 较小时，构件发生弹塑性屈服，极限承载力接近全截面塑性承载力 $N _ { _ { p l \circ } }$

当 λ 增大超过临界值时，构件在达到屈服强度前发生弹性或非弹性失稳，承载力急剧下降。

对于圆端形（Round-Ended）这类异形截面，其弱轴和强轴的回转半径差异显著。仅输入 b 和 h 无法准确描述其绕弱轴失稳的风险，必须计算弱轴方向的 λmin。

# 4.2 相对长细比（λ）

相对长细比是 Eurocode 4 (EN 1994-1-1) 和 GB 50936 中用于综合考量几何柔度和材料性能的关键参数。其定义为塑性承载力与弹性临界承载力之比的平方根：

$$
\overline {{\lambda}} = \sqrt {\frac {N _ {p l}}{N _ {c r}}} = \sqrt {\frac {A _ {s} f _ {y} + A _ {c} f _ {c}}{N _ {c r}}}
$$

或者在某些简化形式下表示为：

$$
\bar {\lambda} = \frac {\lambda}{\pi} \sqrt {\frac {f _ {y}}{2 3 5}} (\text {针 对 纯 钢 结 构 类 比})
$$

对于 CFST 组合柱，Eurocode 4 给出的计算更为精细，其中 $N _ { c r }$ 基于有效抗弯刚度

$$
\left(E I\right) _ {e f f} = E _ {s} I _ {s} + 0. 6 E _ {c} I _ {c} \text {计 算 。}
$$

引入原因：

几何长细比 $\lambda$ 无法反映材料强度的影响。例如，两根几何尺寸完全相同的 CFST 柱，一根使用高强钢 $( f _ { y } { = } 6 9 0 ~ \mathsf { M P a } )$ ），另一根使用普通钢（ $( f _ { y } { = } 2 3 5 ~ \mathsf { M P a } )$ ）。高强钢柱的 λ更大，意味着它在达到更高的屈服应力之前，更容易发生弹性失稳。机器学习模型若只获知 $\lambda$ 和 $f _ { y }$ ，需要极大的数据量才能拟合出这种“强度越高，越易失稳”的复杂耦合关系。直接引入 $\overline { { \lambda } }$ 这一物理特征，或许能显著降低模型的学习难度。

物理机理：

λ 是一个归一化的物理量，它直接映射到柱子失稳曲线的横坐标，衡量了构件在“屈服”与“屈曲”竞赛中的相对倾向。

- $\overline { { \lambda } } \leq 0 . 2$ ：构件被视为极短柱，不发生失稳，强度完全发挥，且由于约束效应显著，承载力往往高于 $N _ { { _ { p l \circ } } }$ 。

- $\overline { { \lambda } } { > } 1 . 0$ ：构件发生弹性失稳，承载力由 $N _ { c r }$ 控制，与材料强度关系减弱。

# 5. 局部稳定性参数：宽厚比与正则化宽厚比

对于方形、矩形及圆端形CFST柱，钢管壁板的局部屈曲是制约其承载力和延性的关键因素。基础参数 t 孤立存在时无法反映这一机制。

# 5.1 径厚比与宽厚比 $\left( D / t , B / t \right)$

宽厚比决定了钢板的欧拉临界屈曲应力 $\sigma _ { c r } \propto k \frac { \pi ^ { 2 } E } { 1 2 \left( 1 - v ^ { 2 } \right) \left( b / t \right) ^ { 2 } } \circ$

当 B / t 过大时，钢管壁板在达到屈服强度 $f _ { y }$ 之前就会发生局部鼓曲（向外凸出）。

一旦发生局部屈曲，钢管的轴向刚度退化，且无法有效约束核心混凝土（甚至与混凝土脱粘），导致组合作用失效。

# 5.2 正则化宽厚比（α）

广泛应用于 Eurocode 3 和 GB 50936 15。其定义考虑了材料性质：

$$
\alpha = \frac {b}{t} \sqrt {\frac {f _ {y}}{2 3 5}}
$$

引入原因：

单纯的 B / t 不足以描述局部屈曲风险，因为高强钢需要更严格的宽厚比限制。正则化宽厚比将几何尺寸与材料屈服应变 $\epsilon _ { y } { = } f _ { y } / E _ { s }$ 结合，是一个真正的力学无量纲数。

该参数衡量了钢板在发生局部屈曲前所能发挥的塑性程度。

● $\alpha$ 较小：截面为“紧凑截面”（Compact Section），能发展全塑性铰，约束效应强。

● $\alpha$ 较大：截面为“极薄截面”（Slender Section），受力处于弹性屈曲阶段，需采用“有效宽度法”计算承载力，钢管对混凝土的约束基本丧失 18。

# 6. 偏心受压参数：弯矩与轴力的耦合

# 6.1 偏心率比（e / b 或 e / h）：

源于 Eurocode 4 和 AISC 360 27。

$$
\epsilon = \frac {e}{D} = \frac {M}{N \cdot D}
$$

其中 $e$ 为荷载偏心距， $D$ 为截面受弯方向的边长或直径。

引入原因：

偏心荷载会引入弯矩，导致截面一侧受压加剧，另一侧可能受拉。这是从轴压构件向压弯构件过渡的核心参数。

应力梯度效应： 偏心导致截面应变分布不均匀。受压侧混凝土膨胀大，钢管约束被激活；受拉侧混凝土开裂，钢管受拉但无约束作用。

二阶效应：偏心率越大，P−δ 效应越显著，导致跨中实际弯矩远大于端部弯矩。

承载力随 $e / D$ 的增加呈急剧下降趋势（即 $N - M$ 相关曲线的下降段）。

# 6.2 相对偏心率（e）

$$
\bar {e} = \frac {e \cdot A}{W}
$$

引入原因：

对于不同形状的截面（如圆端形 vs 矩形），同样的 $e / D$ 引起的应力梯度严重程度不同。用截面模量 W 进行归一化更能反映弯曲应力与轴压应力的相对比例。

根据材料力学公式，一个偏心受压构件边缘的最大应力 $\sigma _ { m a x }$ 由两部分组成：

1. 轴压应力： $\sigma _ { a x i a l } = \frac { N } { A }$

2. 弯曲应力： $\sigma _ { b e n d i n g } { = } \frac { M } { W } { = } \frac { N \cdot e } { W }$

对总应力公式提取公因式 $\frac { N } { A }$

$$
\sigma_ {m a x} = \frac {N}{A} + \frac {N \cdot e}{W} = \frac {N}{A} \left(1 + \frac {e \cdot A}{W}\right)
$$

$$
\sigma_ {m a x} = \sigma_ {a x i a l} \cdot (1 + \bar {e})
$$

结论：

e (即 $\frac { e \cdot A } { W } )$ 可以直接计算出，偏心引起的弯曲效应，让最大压应力在纯轴压的基础上增加了多少倍。

例如：

如果是矩形截面， $\frac { A } { W } = \frac { b h } { b h ^ { 2 } / 6 } = \frac { 6 } { h } \mathfrak { c }$ h 。所以 e= e ⋅6 = $\overline { { e } } = \frac { e \cdot 6 } { h } = 6 \cdot \left( e / h \right) _ { \circ }$

如果是圆形截面， AW = π D 2 / 4π D 3 / 32 = 8D 。所以 e = e ⋅ 8D = 8 ⋅ ( e / D )。 $\frac { A } { W } { = } \frac { \pi D ^ { 2 } / 4 } { \pi D ^ { 3 } / 3 2 } { = } \frac { 8 } { D }$ ${ \overline { { e } } } { = } { \frac { e \cdot 8 } { D } } { = } 8 \cdot \left( e / D \right) _ { \circ }$

# 6.3 偏心率比 $( e _ { 1 } / e _ { 2 } )$

模型中仅仅输入e和 $e / h$ 是不够的，当 CFST 柱面对双曲率弯曲（ $\bf \Pi  ( e _ { 1 } = - e _ { 2 } )$ ）和单曲率弯曲（ $( e _ { 1 } { = } e _ { 2 } )$ ），其承载力完全不同。所以有必要引入偏心率比（ $( e _ { 1 } / e _ { 2 } )$ ），以便模型正确区分出 CFST 柱是呈“C”型弯曲还是“S”型弯曲。
