# CFST字段、特征与参数计算说明

本文档合并了原 `CFST字段与特征说明.md` 与 `参数计算准则.md`，用于统一说明：

- 当前项目中 CFST 原始字段与派生特征的含义
- 几何与等效参数的计算口径
- 当前主线与历史“18特征”结论之间的关系

需要特别说明：

- 当前运行行为仍以 `scripts/compute_feature_parameters.py`、`config/config.yaml`、`train.py`、`predict.py` 和 `src/` 中实际代码为准。
- 文档中的“18特征”结论属于历史经验总结，不等同于当前默认训练配置的硬性定义。

## 1. 当前主线说明

- 当前默认主线已经改为：
  - 报告目标：`Nexp (kN)`
  - 训练目标：`r = (Nexp - Npl) / Npl`
  - 默认训练变换：关闭
- 当前代码会由 `scripts/compute_feature_parameters.py` 离线生成并保留以下派生列：
  - `Npl (kN)`
  - `eta_u = Nexp / Npl`（仅当样本存在 `Nexp (kN)` 时可持久化输出）
  - `r = eta_u - 1`（仅当样本存在 `Nexp (kN)` 时可持久化输出）
  - `b/h`
  - `L/h`
  - `axial_flag`
  - `section_family`
- 其中 `Npl (kN)` 与 `section_family` 当前更偏向目标构造、诊断分析和后续消融，不等同于必须进入默认训练特征集。

## 2. 历史结论摘要

历史结论是：剔除部分绝对几何参数后，仅用一组更稳定的特征表达也可以获得接近甚至相同的预测精度，这支持多截面统一预测模型的可行性。

这条结论今天仍有参考价值，但应视为经验总结，而不是当前代码主线的固定真理。

### 3.2 基础输入参数

这些参数通常直接来自原始试验数据，无需通过公式推导：

1. `b`：截面宽度（mm），定义为 `b >= h`
2. `h`：截面高度（mm）
3. `r0`：钢管外圆角半径（mm）
4. `t`：钢管壁厚（mm）
5. `R`：再生粗骨料取代率（%）
6. `fy`：钢材屈服强度（MPa）
7. `fc`：混凝土抗压强度（MPa）
8. `L`：构件长度（mm）
9. `e1`：上端偏心距（mm）
10. `e2`：下端偏心距（mm）
11. `Nexp`：极限承载力试验值（kN）

## 4. 几何与等效参数计算

### 4.1 辅助变量

- 内圆角半径：`r1 = ((h - 2t) / h) * r0`
- 角径比：`r0/h`

### 4.2 主要几何与等效参数

1. `r0/h`
2. `b/t`
3. `Ac = (b-2t)(h-2t) - (4-π)r1²`
4. `As = 2t(b+h-4t) + (4-π)(r0²-r1²)`
5. `Re = sqrt(Ac/π)`
6. `te = sqrt((Ac+As)/π) - Re`

### 4.3 约束与材料复合参数

1. `ke = 1 - [tan(alpha)(b-2r1-2t)² + tan(beta)(h-2r1-2t)²] / (3Ac)`
   - `tan(alpha) = (h-2t) / (b-2r1-2t)`
   - `tan(beta) = (b-2t) / (h-2r1-2t)`
2. `xi = As * fy / (Ac * fc)`
3. `sigma_re = ke * (te / Re) * fy`

### 4.4 稳定性参数

1. `lambda = L / r`
2. `lambda_bar = sqrt(Npl / Ncr)`

其中：

- `Npl (kN) = (As * fy + Ac * fc) / 1000`
- `Ncr = π² (Es * Is + 0.6 * Ec * Ic) / L²`
- `Es = 206000 MPa`
- `Ec = 4700 * sqrt(fc)`

### 4.5 偏心受压参数

1. `e = max(|e1|, |e2|)`
2. `e/h`
3. `e1/e2`
4. `e_bar = e * A / W`

其中：

- `A = Ac + As`
- `W` 为组合截面在弯曲方向的截面模量

## 5. 当前代码新增的派生列

为了支持 `eta_u / r / Npl` 主线、样本级分层和 regime 诊断，当前代码会额外生成以下派生列：

1. `Npl (kN) = (As * fy + Ac * fc) / 1000`
2. `eta_u = Nexp / Npl`
3. `r = eta_u - 1 = (Nexp - Npl) / Npl`
4. `b/h`
5. `L/h`
6. `axial_flag`
   - `e_bar = 0` 时记为 `axial`
   - 否则记为 `eccentric`
7. `section_family`
   - `circular`
   - `square`
   - `rectangular`
   - `obround`

当前 `section_family` 优先基于原始数据中的 `Group` 列判定：

- `A -> square/rectangular`（再按 `b == h` 区分）
- `B -> circular`
- `C -> obround`

仅在缺少 `Group` 时回退到几何关系判定。

## 8. 惯性矩计算补充

如果需要复核 `Ic / Is` 的弱轴惯性矩计算，可继续参考此前整理的组合法公式：

- `Ic = Ix(b - 2t, h - 2t, r1)`
- `Is = Ix(b, h, r0) - Ic`

其中 `Ix(B, H, R)` 通过圆角矩形的中间矩形、上下翼缘矩形和四个角部 1/4 圆叠加得到。
